شغّل خطوات التحقق المعتمدة في هذا المشروع بالترتيب وأبلغني بأي فشل:

1. `npm run lint`
2. `npm run type-check`

إذا أحدهما فشل: لا تحاول إصلاحه بصمت — اطبع المخرجات الكاملة وأشر للملفات المعنية برابط `[file](path#Lline)` ثم انتظر توجيهي.

إذا نجح الاثنين: أبلغ بنتيجة موجزة (سطر واحد لكل خطوة) ولا تشغّل `npm run build` إلا لو طلبت ذلك صراحة.
